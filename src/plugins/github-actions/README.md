# GitHub Actions

## Enabled

This plugin is enabled when a `.yml` file is found in the `.github/workflows` folder.

## Default configuration

```json
{
  "github-actions": {
    "config": [".github/workflows/*.yml"]
  }
}
```

Also see [Knip plugins][1] for more information about plugins.
